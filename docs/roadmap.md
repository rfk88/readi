# Readi Milestones

## Milestone 0 – Foundations (Weeks 0-2)
- Repo scaffolding, CI, linting
- Google Workspace sandbox + Slack workspace access
- Manual concierge briefs for research validation

## Milestone 1 – Calendar & Queue (Weeks 2-4)
- Google Calendar sync + webhook fallback
- Meeting queue API + basic dashboard
- Participant identity resolution prototype

## Milestone 2 – Gmail & Slack Ingestion (Weeks 4-6)
- Secure message fetching, metadata storage, embeddings
- Data privacy guardrails and rate limit handling

## Milestone 3 – Profile & Context Engine (Weeks 6-8)
- AE/job seeker onboarding, profile tiles CRUD
- Contact overlays, notes, context graph

## Milestone 4 – AI Orchestration (Weeks 8-11)
- Retrieval + research workflows
- Multi-model LLM stack, prompts, validation
- Structured brief output with citations

## Milestone 5 – Delivery & Feedback (Weeks 11-13)
- In-app brief viewer, ratings, notes
- Email + WhatsApp notifications
- Daily prep digest

## Milestone 6 – Security & Controls (Weeks 13-15)
- Connections dashboard, revoke, retention
- Audit logging, incident playbook

## Milestone 7 – Beta Prep (Weeks 15-18)
- Automated tests + evaluation harness
- Internal beta rollout, polish, docs
